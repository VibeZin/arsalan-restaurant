# আর্সালান — Arsalan Nehari & Kabab Corner

A modern restaurant website for **Arsalan Nehari & Kabab Corner**, serving authentic Pakistani & Afghani cuisine in Lalbagh and Wari, Old Dhaka.

Built with React + Vite + TypeScript + Tailwind CSS.

---

## Features

- Full menu with category tabs (Kebabs, Mains, Bread, Drinks)
- Signature dishes showcase with image gallery
- Customer testimonials carousel
- Our Story section with restaurant stats
- Location & contact info for both branches
- Responsive design — mobile-first
- Smooth animations with Framer Motion
- Ember particle effects on hero
- Lightbox gallery viewer

---

## Run Locally

**Prerequisites:** Node.js 18+

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Build for Production

```bash
npm run build
npm run preview
```

---

## Project Structure

```
src/
  components/
    layout/       # Navbar, Footer, MobileMenu
    sections/     # Hero, Signatures, MenuSection, Gallery, etc.
    ui/           # Reusable UI components
  data/           # Menu, gallery, testimonials, contact data
  types/          # TypeScript interfaces
  lib/            # Utility functions
public/
  media/          # Images: dishes/, gallery/, logo/
  fonts/          # Self-hosted font files
```
